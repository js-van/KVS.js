KVS.js

KVS.js is a JavaScript 3D visualization library based on Kyoto Visualization Sysytem (KVS) implemented in C++.

Examples

Here are simple examples with KVS.js. A WebGL enabled browser is required to open the following examples.
  1. Isosurfaces for rectilinear volume data
  2. Isosurfaces for irregular volume data (tetrahedral mesh)
  3. Streamlines
